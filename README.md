# Epifanía del Café — versión local

## Requisitos
- Node.js 18 o superior

## Cómo correrlo
1. Abrí una terminal en esta carpeta
2. Instalá dependencias:
   npm install
3. Iniciá el entorno local:
   npm run dev
4. Abrí en el navegador la URL que te muestre Vite

## Qué incluye
- Título superior: EPIFANÍA DEL CAFÉ
- 12 cartas con números romanos
- Dorso único para todas las cartas
- Flip al hacer click
- 3 frentes cargados:
  - Alquimista
  - El Café
  - El Método
- 9 placeholders listos para reemplazar cuando tengas las demás cartas
