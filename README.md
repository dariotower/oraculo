# Epifanía del Café — HTML estático con modal

Esta versión es HTML/CSS/JS puro.
No usa React ni necesita build.

## Cómo verlo local
Abrí `index.html` en el navegador.

## Incluye
- 12 cartas
- dorso único
- modal con fondo desenfocado
- animación de entrada
- flip dentro del modal
- 3 cartas cargadas
- 9 placeholders
